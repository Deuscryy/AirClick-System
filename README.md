# 👆 AirClick - Sistema de Interfaz Sin Contacto

AirClick es una herramienta de accesibilidad y productividad desarrollada en Python que permite a los usuarios abrir aplicaciones y ejecutar comandos en Windows utilizando únicamente gestos de la mano a través de una cámara web.

## 🚀 Características Principales
* **Reconocimiento en tiempo real:** Utiliza Inteligencia Artificial para mapear los puntos de las manos.
* **Sistema de Estados (Modos):** Capacidad de ciclar entre diferentes modos de uso usando un solo dedo, multiplicando la cantidad de comandos disponibles.
* **Seguro de Gesto y Cooldown:** Algoritmos de protección para evitar la apertura múltiple de aplicaciones y saturación del sistema operativo.
* **Interfaz Gráfica Integrada:** Panel de control moderno con feedback visual en vivo.

## 🛠️ Tecnologías Utilizadas
* **Python** (Lenguaje principal)
* **OpenCV** (Captura y procesamiento de video)
* **MediaPipe** (Modelo de IA para tracking de manos)
* **CustomTkinter** (Diseño de la Interfaz Gráfica GUI)
